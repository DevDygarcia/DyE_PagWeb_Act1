Tema 1: Elementos de Páginas Web

I. Orígenes de la Web

Los orígenes de la Web se remontan al CERN (Centro Europeo de Física Nuclear) en Ginebra, Suiza. Tim Berners-Lee, un ingeniero y físico británico, creó la World Wide Web (WWW) en 1991. La WWW consiste en un conjunto de protocolos que operan sobre Internet y permiten el acceso flexible y generalizado a la información almacenada en la red en diversos formatos.

  • Web 2.0: Se refiere a la evolución de la Web hacia una plataforma más interactiva y colaborativa, donde los usuarios pueden generar contenido y participar activamente.
  • Web 3.0: Aunque no existe una definición precisa, se relaciona con la Web semántica, donde las máquinas pueden comprender y procesar datos de manera más inteligente.
  • La W3C y los estándares: La World Wide Web Consortium (W3C) establece estándares para la Web, como HTML, CSS y otros protocolos.

II. Lenguajes de etiquetas

  • SGML (Standard Generalized Markup Language): Un lenguaje base para definir otros lenguajes de marcado.
  • XML (Extensible Markup Language): Utilizado para estructurar datos y compartir información entre sistemas.
  • HTML (HyperText Markup Language): El lenguaje fundamental para crear páginas web.
  • XHTML (Extensible HyperText Markup Language): Una versión más estricta y compatible con XML de HTML.
  • HTML5: La última versión de HTML, con mejoras en multimedia, semántica y accesibilidad.

III. Otros lenguajes relacionados al desarrollo web

  • CSS/CSS3: Para estilizar y diseñar páginas web.
  • JavaScript: Lenguaje de programación utilizado para crear interactividad y dinamismo en las páginas.
  • Ajax: Tecnología para cargar datos de forma asíncrona sin recargar la página completa.
  • Tecnologías y lenguajes del lado del servidor: Ejemplos incluyen PHP, Python, Ruby, etc.
  • Navegadores: Software que interpreta y muestra páginas web.
  • Motors renderizados: Componentes que procesan y muestran el contenido HTML y CSS en los navegadores.

IV. Herramientas de desarrollo

  • Software WYSIWYG: Editores visuales que muestran cómo se verá la página mientras se trabaja en ella.
  • Figma, InVision, Zeplin: Herramientas para diseño y colaboración en proyectos web.
  • GitHub: Plataforma para gestionar versiones y colaborar en proyectos de desarrollo.
  • Inspección de código desde el navegador: Permite examinar y depurar el código directamente en el navegador.

Tema 2: El Diseño De Una Página Web

  • Tipos de páginas Web: Pueden ser informativas, comerciales, blogs, etc.
  • Puntos básicos en la elaboración de una página Web: Considerar la estructura, el diseño, la usabilidad y la experiencia del usuario.

Tema 3: Estructura De Una Página Web

  • Jerarquía de directorio de un proyecto web: Organización de archivos y carpetas.
  • Elementos de estructura para el diseño de una página Web:
    • Regla utilizada para validar y certificar una página web: Cumplimiento de estándares y buenas prácticas.
    • Reglas para escribir HTML: Sintaxis y etiquetas correctas.
    • Estructura básica de un archivo web: Encabezado, cuerpo y pie de página.
    • Etiquetas: Al menos 40 etiquetas HTML.
    • Atributos: Propiedades que modifican el comportamiento de las etiquetas.
    • Elemento HTML: Componentes como encabezados, párrafos, imágenes, etc.
    • Elementos en línea y en bloque: Diferencias entre elementos que afectan el flujo del contenido.
